# SlideBlocks — PPT 智能组装 Skill

告诉 AI 你要做什么样的 PPT，它会从素材库里搜页面、套模板、组装出成品。

---

## 环境要求

- Windows 系统
- Microsoft PowerPoint 或 WPS（必须安装）
- Python 3.9+

```bash
pip install anthropic python-pptx pyyaml flask pywin32
```

---

## 第一次使用（配置两步走）

### 第一步：填写 config.yaml

打开 skill 文件夹里的 `config.yaml`，填写两个路径：

```yaml
materials_dir: "素材文件夹的路径"   # 问维护人要，可以是网络共享盘路径
output_dir:    "你本地的输出文件夹"  # 组装好的 PPT 放在这里
```

### 第二步：运行路径初始化脚本

```bash
python setup_paths.py
```

这会把数据库里的路径更新成你机器上的实际路径，只需运行一次。

## 怎么用

在 AI 对话里直接说你的需求：

**组装新 PPT**
> "帮我做一个售前汇报 PPT，客户是医院，主题是 AI 辅助诊断，需要背景介绍、产品功能、案例三个章节"

> "帮我组装一份行业会议演讲稿，聚焦数字化转型，要科技风深色模板"

**搜索素材**
> "帮我找几张关于数据平台的幻灯片"

> "搜一下有没有讲 AI 病历生成的页面，要质量高的"

**调整已有 PPT**
> "把刚才输出的 PPT 第 3 页删掉"

> "在第 5 章前面插入一个过渡页，标题写'解决方案'"

> "把第 10 页换成哈密那份 PPT 的第 15 页"

## 模板

`模板/` 文件夹里有两套默认模板，AI 会列出让你选：
- **科技风（深色底）** — 深色背景，适合 AI / 数字化方向
- **蓝色商务（浅色底）** — 白色背景，通用商务风格

想加自己的模板，见 `模板/模板说明.md`。

---

## 素材库有更新怎么办

让维护人把新的 `slide_vault.db` 发给你，替换掉 skill 文件夹里的同名文件，再重新运行一次 `setup_paths.py` 即可。

---

## 常见问题

**组装时 PowerPoint 窗口一直弹出来？**
正常现象，完成后自动关闭。期间不要手动点 PowerPoint 窗口。

**搜索结果不理想？**
换几个同义词试试，或告诉 AI "换个关键词再搜一次"。

**电脑同时装了 WPS 和 Office？**
会优先调用 Office，只有 WPS 也能正常用。
